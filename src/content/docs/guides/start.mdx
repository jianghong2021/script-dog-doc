试着快速体验脚本狗，从创建一个新的项目(或导入已有的)开始，完成编程后运行起来。
<Aside type='caution' title='使用前提'>
打开浏览器的`开发者模式`(**必须**)，如果开启后依然提示 **请启用开发者模式** ，请禁用此插件后再次启用。

有以下方法：
    - 浏览器地址栏输入`chrome://extensions/`(Chrome浏览器)，`edge://extensions/`(Edge浏览器)
    - 浏览器`设置>扩展程序`
    - 浏览器右上角通常有`扩展程序`管理页面的快捷按钮

`开发者模式`的开关按钮位置，Chrome浏览器在右上角，Edge浏览器在左下角，按钮位置可能会随着版本变化。
</Aside>
<Steps>
1. #### 管理页面
    - 点击插件图标，或右键菜单找到*脚本狗*
2. #### 填写项目信息
    - 名称 - 项目名称
    - 描述 - 项目描述
    - 类型 - *内容脚本*，运行于指定的网页中，用于完全自定义当前网页，可以访问所有脚本狗功能。*独立窗口*，将脚本运行在一个隔离窗口中。*侧边栏*，和独立窗口一致。
        <Aside>
            目前*独立窗口*，*侧边栏*运行于隔离沙盒可以访问所有脚本狗模块功能，但不能访问用户设备(录音，录像)
        </Aside>
    - 脚本狗模块 - 如果使用了脚本狗的模块需要勾选上。
    - GM模块 - 如果是导入的GM脚本(篡改猴等)默认会勾选。
    - 项目窗口 - 类型为*独立窗口*时才会有效，配置独立窗口启动时的位置，尺寸等。
    - 内容脚本 - 类型为*内容脚本*有效
        - 匹配 - 指定内容脚本运行于什么样的网页，以网址进行匹配，默认`*://*/*`,匹配所有网页。如：`https://*.baidu/*` 匹配百度旗下所有网站，`https://tieba.baidu.com/*`仅匹配贴吧。
        - 运行时机 - 指定脚本在什么时候运行
            - 文档开始 - 网页自身所有脚本之前运行，此时可能还不能操作元素。
            - 文档结束 - 网页准备就绪后，此时网页自身一些资源(图片，视频等)尚未加载完成。
            - 文档空闲 - 网页一切准备就绪，并闲暇之余。此时所有操作均已可行。
3. #### 编辑项目
    - 删除 - 从本地删除，不影响云上的。
    - 编辑 - 修改项目信息
    - 模板 - 为项目配置模板，通过代码创建元素过于麻烦？那就是试试它！
    - 编码 - 脚本开发，默认为blockly可视化脚本，如果你有编程经验也可以选择文本编程模式。
    - 运行 - 运行*独立窗口*，*侧边栏*，*内容脚本*不需要手动运行，仅需要打开匹配的网页即可。
    - 下载 - 将项目打包下载到磁盘，用于分享给他人导入。
3. #### 导入项目
    - 脚本狗自身项目，由上面*下载*得到的xxx.json文件。
    - GM脚本，由第三下载得到的油猴脚本xxx.js文件。
4. #### 刷新项目
    - 仅针对内容脚本，当脚本修改后需要刷新才能生效。
</Steps>
