試著快速體驗腳本狗，從創建一個新的項目(或導入已有的)開始，完成編程後運行起來。
<Aside type='danger' title='使用前提'>
開啟瀏覽器的`開發者模式`（必須）

有以下方法：  
   - 在瀏覽器地址欄輸入`chrome://extensions/`（Chrome瀏覽器）、`edge://extensions/`（Edge瀏覽器）。  
   - 前往瀏覽器的`設定 > 擴充功能`。  
   - 使用瀏覽器右上角通常提供的`擴充功能`管理頁面快捷按鈕。  

`開發者模式`的開關按鈕位置：Chrome 瀏覽器位於右上角，Edge 瀏覽器位於左下角，但按鈕位置可能會隨著版本變化。
</Aside>

<Steps>
1. #### 管理頁面
    - 點擊插件圖標，或右鍵菜單找到*腳本狗*
2. #### 填寫項目信息
    - 名稱 - 項目名稱
    - 描述 - 項目描述
    - 類型 - *內容腳本*，運行於指定的網頁中，用於完全自定義當前網頁，可以訪問所有腳本狗功能。*獨立窗口*，將腳本運行在一個隔離窗口中。*側邊欄*，和獨立窗口一致。
        <Aside>
            目前*獨立窗口*，*側邊欄*運行於隔離沙盒可以訪問所有腳本狗模塊功能，但不能訪問用戶設備(錄音，錄像)
        </Aside>
    - 腳本狗模塊 - 如果使用了腳本狗的模塊需要勾選上。
    - GM模塊 - 如果是導入的GM腳本(篡改猴等)默認會勾選。
    - 項目窗口 - 類型為*獨立窗口*時才會有效，配置獨立窗口啟動時的位置，尺寸等。
    - 內容腳本 - 類型為*內容腳本*有效
        - 匹配 - 指定內容腳本運行於什麼樣的網頁，以網址進行匹配，默認`*://*/*`,匹配所有網頁。如：`https://*.baidu/*` 匹配百度旗下所有網站，`https://tieba.baidu.com/*`僅匹配貼吧。
        - 運行時機 - 指定腳本在什麼時候運行
            - 文檔開始 - 網頁自身所有腳本之前運行，此時可能還不能操作元素。
            - 文檔結束 - 網頁準備就緒後，此時網頁自身一些資源(圖片，視頻等)尚未加載完成。
            - 文檔空閒 - 網頁一切準備就緒，並閒暇之餘。此時所有操作均已可行。
3. #### 編輯項目
    - 刪除 - 從本地刪除，不影響雲上的。
    - 編輯 - 修改項目信息
    - 模板 - 為項目配置模板，通過代碼創建元素過於麻煩？那就是試試它！
    - 編碼 - 腳本開發，默認為blockly可視化腳本，如果你有編程經驗也可以選擇文本編程模式。
    - 運行 - 運行*獨立窗口*，*側邊欄*，*內容腳本*不需要手動運行，僅需要打開匹配的網頁即可。
    - 下載 - 將項目打包下載到磁盤，用於分享給他人導入。
3. #### 導入項目
    - 腳本狗自身項目，由上面*下載*得到的xxx.json文件。
    - GM腳本，由第三下載得到的油猴腳本xxx.js文件。
4. #### 刷新項目
    - 僅針對內容腳本，當腳本修改後需要刷新才能生效。
</Steps>
